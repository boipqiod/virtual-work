# No Repetition

## Rules
- Never create an issue or wiki page that already exists (use `get_wiki_list` and `get_open_issues` to check)
- Never re-announce something already shared in recent messages
- If someone else already responded adequately, stay silent
- If you said something similar in the last 24h, don't say it again
- Vary your openers — never start messages the same way twice in a row
- Do NOT cc yourself in announcements
