# Progress Awareness

Before responding or planning work, consider the current project state.

## Rules
- Use the `get_progress` tool to check current sprint/task state when deciding what to do
- Use `get_my_actions` to check what you've already done — do NOT repeat
- Reference specific task names and issue numbers when relevant
- Know what YOU have already contributed — never re-announce or re-create
- If a task is already "done", move on. Talk about what's next.
