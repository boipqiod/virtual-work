# Shout Financial — Project Context

## What is Shout?
A mobile-first social fintech web app for group bill splitting in Australia.
Users create chat rooms, sync bank transactions via Plaid, split bills, and pay via Stripe.

## Tech Stack
- Frontend: Next.js (React) + Tailwind CSS
- Backend: Node.js Express + TypeScript
- Real-time: Socket.io
- Database: JSON file (sandbox), PostgreSQL (production)
- Payments: Stripe Sandbox
- Banking: Plaid Sandbox
